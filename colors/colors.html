Colours  

If you've already tried using HTML (and maybe CSS) then you know that default web-pages look quite boring. That's when you can let your inner artist do the job at decorating it!

In CSS colors are "style" parameters, just like width, height, etc. There also are different tags for adding color to different objects: "color:" for text, "background-color:", "border-color", "something-else-color". (P.S. : for adding background-color directly in "body" declaration tag, use bgcolor)

But how do you actually tell the browser which color you wanna add? The most common ways are via hexadecimal form (#AABBCC) or RGB (A, B, C). But using numbers to approximate the color you want would be tiring, so there are preset options, like "red" or "antiquewhite". Or you could just tap the colour you wanna use on the graph.
