What are Divs?  

Divs are divisions (sections, containers) of a web-page. They are declared with use of "div" tag, can contain any type of information (text, pictures, videos, audios) and are easily changeable.

Most common parameters for a div are:

  • width   and   height
  • color
  • background-color:
  • position

Width and height are responsible for the size of your container, therefore are often measured in pixels (px). Other options include viewport width (vw) and percentages (%)

As you already know from previous page, color and background-color take in color code values (usually in RGB or HEX).

Position is the more interesting of these. It has a few values, such as:

static
The default value; does nothing new with the div
relative
Usually comes with a set of left: right: top: bottom: offsets, which move the div from its postion as if it were static.
absolute
Hovers over the rest of the content. Also comes with offsets! The offsets position it relative to its container.
sticky
Always on page, even if you try to scroll past it
inherit
Gets whatever value the "parent" had
