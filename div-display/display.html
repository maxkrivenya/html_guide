  Divs: Display  

Display is a div property that determines the type of box(es) created for an element. Works in combination with float and position.

flex

Adds additional axes to position your items at! Before that, you would have to stuff everything in one row. Now, using justify- and align- properties you can position items in relation to:

  • div's   borders (left, right, top, bottom)
  • div's   central axes (center)
  • other elements within   the div   (space-between/-around/-equally)

Also you can change how the items are directed using flex-direction: row/column/row-reverse/column-reverse.



grid

Cool stuff! Imagine a sudoku but without the numbers. That's basically a 9x9 grid with equal square tiles. You can set the amount of rows and columns yourself using grid-template-rows and grid-template-columns. You can set width and height with "px", "%" and a new unit - fraction units! ("fr"). These can be used to equally divide up the leftover space. For example, grid-template-columns: 1fr 500px 1fr would give you a middle column of 500px and one column on each side takind up the rest of the grid width.
